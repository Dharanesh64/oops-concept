## 🧩 Topics Covered

### 🔌 Interface

* Defines **what a class should do**
* Supports **multiple implementations**
* Helps in **flexible and scalable design**

---

### 🧬 Inheritance

* One class **inherits properties and behavior** of another
* Promotes **code reuse**
* Supports hierarchy (Parent → Child)

---

### 🔁 Polymorphism

* Same method → **different behavior**
* Types:

  * Compile-time (**Method Overloading**)
  * Runtime (**Method Overriding**)

---

### 🔒 Encapsulation

* Wraps **data + methods** into one unit
* Uses **access specifiers**
* Ensures **data hiding and security**

---

### 🎯 Abstraction

* Hides **implementation details**
* Shows only **essential features**
* Achieved using:

  * Interfaces
  * Abstract Classes
